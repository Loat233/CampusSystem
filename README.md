## CampusSystem - 校园兼职信息管理与应聘平台
###  项目概述
CampusSystem 是一个基于Java开发的校园兼职信息管理与应聘平台。本系统为校园内的学生和用人单位之间搭建了一个便捷的信息桥梁，实现了兼职岗位的发布、申请、审核和管理等全流程线上化。

**核心角色**：
- **学生端** : 浏览兼职信息，提交应聘申请，查看申请进度
- **管理员端** : 管理所有兼职岗位，审核学生申请，维护系统运行
**版本演进**：项目已从早期控制台版本升级为功能完善的图形界面版本，提供了更直观的用户体验。

### 核心功能

| 学生端功能     | 功能描述                               |
| --------- | ---------------------------------- |
| **用户登录**​ | 通过学号和密码登录系统                        |
| **浏览兼职**​ | 表格形式查看所有可申请的兼职岗位，包含岗位详情、公司信息、薪资待遇等 |
| **提交申请**​ | 选择心仪岗位，一键提交应聘申请，系统自动记录申请时间         |
| **申请跟踪**​ | 实时查看个人申请状态（待审核/已通过/未通过）            |

| 管理员端功能     | 功能描述                |
| ---------- | ------------------- |
| **管理员登录**​ | 管理员专属登录入口           |
| **岗位管理**​  | 新增、编辑、删除兼职岗位信息      |
| **审核申请**​  | 查看所有学生申请，进行录用/不录用处理 |
| **数据查看**​  | 浏览所有岗位和申请记录，支持详情查看  |
### 技术栈概览
* 开发语言 : Java
* 用户界面 : Swing (GUI)
* 数据存储 : Java对象序列化

### 启动系统
- **学生端启动** : 运行 `src/gui/UserLogin.java`
- **管理员端启动** : 运行 `src/gui/AdminLogin.java`
- **控制台版本** : 运行 `src/core/Login.java`

### 账户系统 (AccountSys)
- **功能**: 用户身份认证
- **实现**:
    - 从 `accounts.txt`读取账户信息
    - 区分学生账户（纯数字ID）和管理员账户（ID前带\*号）
    - 提供 `isCorrectStud()`和 `isCorrectAdmin()`验证方法

### 数据存储 (DataBase)
- **存储机制**: Java对象序列化
- **文件结构**:
    - `jobs`- 存储所有岗位对象列表
    - `applications`- 存储所有申请记录列表
- **核心方法**:
    - `saveJobs()`/ `loadJobs()`- 岗位数据持久化
    - `saveApplications()`/ `loadApplications()`- 申请数据持久化
    - 增删改查(CRUD)操作方法

### 图形界面
* 学生界面 (UserFrame)
	1. **岗位浏览**: 表格展示所有岗位信息
	2. **申请提交**: 选择岗位后点击申请按钮
	3. **进度查看**: 表格展示个人申请状态
* 管理员界面 (AdminFrame)
	1. **岗位管理**: 列表展示，支持增删改查
	2. **申请管理**: 查看所有申请，进行审核操作
	3. **详情查看**: 右侧面板显示选中项的详细信息
